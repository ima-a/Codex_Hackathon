# Inventory HMARL

Hierarchical Multi-Agent Reinforcement Learning for Inventory Management

## Project Structure

- `config/` - Simulation configuration
- `data/generated/` - Synthetic demand data
- `demand/` - Demand generation and forecasting
- `env/` - Digital twin environment
- `agents/` - Multi-agent system (store, warehouse, coordinator)
- `reconciliation/` - Reward engine
- `scenarios/` - Scenario definitions
- `simulation/` - Simulation runner
- `evaluation/` - Performance metrics
